# PlotEngine 产品介绍文档

> 面向市场营销团队的产品功能说明

---

## 产品概述

**PlotEngine** 是一款专为创意写作者打造的智能写作平台，集成了先进的 AI 辅助功能，帮助作家更高效地创作小说、剧本和其他长篇故事内容。

### 核心定位
- **目标用户**：小说作家、剧本创作者、网文写手、创意写作爱好者
- **平台支持**：macOS 桌面应用、Web 网页版
- **核心价值**：AI 驱动的智能写作助手 + 专业的故事管理系统

---

## 核心功能

### 1. 智能编辑器

#### 多标签编辑
- 支持同时打开多个章节进行编辑
- 预览标签和永久标签两种模式
- 自动保存功能（5秒防抖）
- 支持 Markdown 格式

#### 实体识别与高亮
- AI 自动识别文本中的角色、地点、物品、事件
- 智能高亮显示，支持悬停查看详情
- 点击实体名称可快速编辑信息

### 2. 知识库管理系统

#### 默认分类
| 分类 | 说明 |
|------|------|
| 📚 章节 | 管理所有故事章节 |
| 👤 角色 | 人物档案管理 |
| 📍 地点 | 场景与位置设定 |
| 📦 物品 | 重要道具记录 |
| 📅 事件 | 关键剧情节点 |

#### 自定义扩展
- 支持创建自定义分类标签
- 可上传自定义图标
- 灵活管理各类故事元素

### 3. AI 智能助手

#### 实时分析功能
系统会在后台自动分析您的写作内容，提供以下建议：

| 功能 | 说明 |
|------|------|
| 🔍 一致性检查 | 检测剧情矛盾和逻辑漏洞 |
| 💡 伏笔建议 | 提供前后呼应的写作建议 |
| ⏰ 时间线验证 | 确保故事时间线的连贯性 |
| 🎭 角色发展 | 分析角色塑造的合理性 |

#### AI 写作辅助
- **问答功能**：向 AI 询问关于故事的任何问题
- **续写功能**：AI 根据上下文自动续写内容
- **修改功能**：选中文本后让 AI 帮助润色或改写

#### 智能触发机制
- 新文件打开 10 秒后自动分析
- 每写作约 200 字自动触发分析
- 可随时手动开启/关闭后台分析

### 4. 多语言支持

#### 界面语言

#### 智能语言检测
- 首次启动自动检测系统语言
- AI 回复会根据用户语言设置自动切换

### 5. 界面自定义

#### 面板折叠功能
- AI 助手面板可折叠/展开
- 知识库面板可折叠/展开
- 折叠状态自动保存

#### 主题设置
- 浅色主题
- 深色主题
- 跟随系统

## 技术亮点

### AI 能力
- 基于 Claude (Anthropic) 大语言模型
- 支持上下文理解和故事连贯性分析
- 多语言 AI 响应

### 数据安全
- 桌面版：本地文件存储，数据完全掌控
- Web 版：云端安全存储，Google OAuth 认证

### 性能优化
- 防抖自动保存，避免频繁写入
- 智能分析触发，节省 API 调用
- 响应式界面，流畅用户体验

## 使用场景

### 场景一：网络小说创作
> 作者正在创作一部百万字的玄幻小说，使用 PlotEngine 管理复杂的人物关系和世界观设定，AI 自动检测前后矛盾，避免"挖坑不填"。

### 场景二：剧本写作
> 编剧使用 PlotEngine 管理多条故事线，AI 伏笔建议帮助构建精巧的剧情反转，时间线验证确保叙事顺序合理。

### 场景三：创意写作教学
> 写作课程教师使用 PlotEngine 演示专业写作流程，学生可以直观理解故事结构和角色发展。
