The Halloween theme is built on Material Design 3 with custom color schemes:
- Uses dark mode as the base
- Custom color palette with orange, purple, and green accents
- Animated decorations using Flutter's CustomPainter
- Persistent theme selection saved to local storage

## Customization

The theme is defined in `lib/config/app_themes.dart` and can be easily customized by modifying the color values in the `halloweenTheme` getter.
